Internship Opportunities

Robotics Software Intern Internship

Work on cutting-edge autonomous mobile robot systems, implementing navigation algorithms and sensor fusion.

Prerequisites:

  • Strong mathematics background (Linear Algebra and Geometry)
  • Proficiency in Python programming
  • Understanding of robotics fundamentals
  • Familiarity with ROS/ROS2 (preferred)

What you'll learn:

  • SLAM algorithms and path planning
  • Sensor integration (LiDAR, IMU, Cameras)
  • Real-world robotics system development

Computer Vision Intern Internship

Develop AI-powered vision systems for object detection, tracking, and 3D reconstruction in industrial applications.

Prerequisites:

  • Python programming experience
  • Basic understanding of deep learning
  • Familiarity with OpenCV (preferred)
  • Knowledge of CNN architectures

What you'll learn:

  • Training and deploying deep learning models
  • 2D/3D vision system integration
  • Real-time image processing pipelines

Embedded Systems Intern Internship

Design and develop embedded control systems for robotic platforms, working with microcontrollers and real-time systems.

Prerequisites:

  • C/C++ programming skills
  • Understanding of microcontroller architectures
  • Basic electronics and circuit design knowledge
  • Experience with Arduino/STM32 (preferred)

What you'll learn:

  • Real-time embedded systems development
  • Motor control and sensor interfacing
  • Hardware-software integration

Full-Time Positions

Senior Robotics Engineer Full-Time

Lead the development of autonomous mobile robot systems from concept to deployment.

Requirements:

  • 5+ years of experience in robotics development
  • Expert-level knowledge of ROS/ROS2
  • Strong background in SLAM, path planning, and control systems
  • Experience with sensor fusion and localization
  • Proficiency in C++ and Python

Machine Learning Engineer Full-Time

Develop and deploy machine learning models for computer vision and autonomous decision-making systems.

Requirements:

  • 3+ years of ML/AI development experience
  • Strong knowledge of PyTorch or TensorFlow
  • Experience with computer vision and object detection
  • Understanding of model optimization and deployment
  • Publication record (preferred)
